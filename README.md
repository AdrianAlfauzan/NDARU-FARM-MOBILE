### 👨‍💻 Cara Menjalankan Proyek

1. Clone repositori ini ke direktori web server Anda.

   ```
   git clone https://github.com/BhinnekaDev/NDARU-FARM-MOBILE.git

   ```

2. Jalankan aplikasi menggunakan Expo.

   ```
   cd NDARU-FARM
   npm install
   npx expo start
   ```
